### 1. Установка зависимостей
```bash
npm install
```

### 2. Запуск бэкенда
```bash
npm run backend
```
- Эндпоинт категорий: `http://localhost:3000/categories`
- Эндпоинт таблицы лидеров: `http://localhost:3000/leaderboard`

### 3. Запуск фронтенда
```bash
npm start
```
- Ссылка на сайт: `http://localhost:4200`
